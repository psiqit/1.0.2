# System Requirements

## Minimum Requirements

- Python 3.9 or higher
- pip package manager

## Core Dependencies

| Package | Version | Purpose |
|---------|---------|---------|
| numpy | ≥1.21.0 | Numerical computations |
| scipy | ≥1.7.0 | Scientific functions & optimization |
| matplotlib | ≥3.4.0 | Plotting (Bloch sphere, Wigner, etc.) |
| sympy | ≥1.9.0 | Symbolic mathematics |

## Optional Dependencies

| Package | Purpose |
|---------|---------|
| jupyter | Interactive notebooks |
| pylatex | Direct PDF generation |
| qiskit | Qiskit compatibility (optional) |

## Install Dependencies


bash
```
pip install numpy scipy matplotlib sympy
```
##For full features:


bash
```
pip install psiqit[full]
```
